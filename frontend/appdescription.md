# Bird Count

Use Bird Count to record th birds you see.

The user interface displays a navigation bar with three tabs and a configuration screen icon on the right

## Home

The home screen has four areas vertically displayed

### Bird Selector

The bird selector shows a list of birds to choose from. Clicking on one brings up an input where you can adjust the number, and cancel or accept.

### Search

The search bar shows a summary of the current filter being applied to the Bird Selector

### Keyboard Input

The keyboard lets you enter in filter criteria that limits the displayed species

## Summary

The summary screen displays the counts of birds you have seen in a selectable time interval broken out by species. It also lets you email the current summary.

## Log

The log screen shows a chronological list of entries. Swipe left to expose a delete button.
